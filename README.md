# MqttClient
This repository demonstrates an Android based application demonstrating MQTT functionalities such as:
1. Connect
2. Disconnect
3. Publish
4. Subscribe
5. Unsubscribe 

It makes use of the following:
* Eclipse Paho - Client
* HiveMQTT Public Broker - Broker
